Computer Programming and Web Design

The Computer Programming and Web Design (CPWD) program at Bayside High School is designed to equip students with essential skills in computer programming, web development, and digital design. This comprehensive program offers hands-on experience in coding languages such as HTML, CSS, JavaScript, and Python, as well as training in web design principles and user experience (UX) design. Students will learn to create and maintain websites, develop software applications, and understand the fundamentals of computer science. The CPWD program aims to prepare students for careers in the rapidly evolving tech industry, providing them with the knowledge and skills needed to succeed in various roles such as web developers, software engineers, and IT specialists. Through a combination of classroom instruction, project-based learning, and real-world applications, students will gain a solid foundation in computer programming and web design, setting them on a path toward future success in the digital age.
